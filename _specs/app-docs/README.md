# Vibe × Symmio Docs Compendium — build spec

Full spec for the `/goal` agent that builds the **unified manifesto of intent-based derivatives** — the joint Symmio × Vibe documentation compendium: 500-800 pages, an **answer-engine front door**, a **living-docs** feedback loop, the best documentation in web3. Runs as **two sessions: research, then implementation.**

**To launch:** paste [`00-goal-prompt.md`](00-goal-prompt.md) into `/goal` (kept under the 4000-char limit; it references the files below). The agent must run from / have access to the `onboarding-app` repo so the relative paths resolve.
