## 3.5 The Protocol That Fills the Gap

The protocol should be **very simple**. No need for complex mechanisms. Let evolution play out with launchpads and creators trying their own things. We already have all the tools. The only missing ingredient:

- The ability to **vote "NO"** on created tokens
- The ability to get **leverage**

**The missing piece**: The ability to create futures for any token directly at launch.
