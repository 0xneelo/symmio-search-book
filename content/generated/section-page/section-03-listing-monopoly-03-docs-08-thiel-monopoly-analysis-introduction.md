## Introduction

Peter Thiel's framework for monopoly businesses provides a rigorous lens for evaluating whether controlling the token lifecycle-and specifically filling the perp gap-constitutes a path to durable monopoly power. This analysis applies Thiel's criteria to the listing monopoly thesis as a **hypothetical category analysis**, not as a claim that any specific protocol already owns this market.

**Current market status**: The category is still structurally open. Major venues dominate mature perpetuals, but no protocol has yet established scaled dominance in permissionless perp creation for the long tail of newly graduated tokens.

**Thiel's central insight**: There are two kinds of businesses—perfectly competitive and monopolies. The goal is to build the latter. Monopolies create unique value and capture it; competitive businesses grind toward zero profit.
