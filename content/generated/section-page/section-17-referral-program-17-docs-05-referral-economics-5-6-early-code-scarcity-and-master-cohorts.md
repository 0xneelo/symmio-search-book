## 5.6 Early-Code Scarcity and Master Cohorts

The transcript also explores an early-cohort model where initial referral code holders receive structurally stronger distribution rights during the access-gated phase.

If used, this model should define:

- exact cohort size and eligibility path,
- duration of enhanced rights,
- propagation depth limits,
- and sunset rules that prevent permanent centralization.

Scarcity can drive growth velocity, but only when paired with transparent issuance and expiry policy.
