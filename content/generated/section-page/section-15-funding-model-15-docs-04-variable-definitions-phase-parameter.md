## Phase Parameter

| Variable | Symbol | Range | Description |
|----------|--------|-------|-------------|
| Counterparty Share | `α` | `[0, 1]` | How much LP is counterparty |

**Phase 1:** `α ≈ 1` — LP/solver is the counterparty
**Phase 2:** `α ≈ 0` — Traders mostly offset each other
