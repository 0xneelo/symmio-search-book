# The Scale Of The Lifecycle Gap

Neelo's scale argument is simple: token creation is abundant, but perp market creation is scarce.

The source contrasts daily token launches and large launchpad throughput with the much smaller number of tokens that ever receive perpetual markets on major venues. Even if the exact counts change, the structural ratio is the important claim. Permissionless issuance created far more assets than existing derivative infrastructure can serve.

The same gap is even sharper in lending. Long-tail tokens rarely have liquid borrowing markets unless they use isolated designs, and those designs still need collateral, risk controls, and users willing to supply liquidity.

## What Scale Changes

Scale changes the problem from listing review to infrastructure design.

If there were only a few new assets, manual venue selection could be enough. When thousands of assets appear and only a tiny fraction can receive mature derivatives, the market needs a filtering and bootstrap layer. Otherwise, price discovery stays concentrated in spot markets and late-stage venues inherit only the already-obvious survivors.

## Publication Boundary

Do not publish the source's launch counts, venue counts, or ratios as current live facts without fresh primary-source review. The source supports the directional scale mismatch: token issuance massively outpaces perp and lending market availability.
