# How To Read The Thiel Scorecard

The final scorecard in Neelo's source is a compact strategy map. It lists value creation, value capture, small-market start, proprietary technology, network effects, economies of scale, brand, last-mover potential, durability, vertical integration, and avoiding competition.

The scorecard should not be published as an audited investment conclusion. It is a way to organize the argument. Each row asks for evidence:

- value creation requires markets that could not otherwise exist;
- value capture requires durable economics, not only activity;
- small-market start requires a clearly owned wedge;
- proprietary technology requires a capability gap;
- network effects require real participants reinforcing each other;
- scale economics require reusable infrastructure;
- brand requires earned trust;
- last-mover potential requires category definition and integration depth;
- durability requires survival across market regimes.

Read this way, the scorecard is useful because it turns "Vibe is different" into a checklist. The compendium can then connect each checklist row to pages, sources, open decisions, and future verification.
