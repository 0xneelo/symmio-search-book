# Vibe Trading Program Points

The official trading-program page describes the pre-TGE trading leaderboard point pool. It says early-access trading earns points used to claim future rewards after TGE, with 100,000 points available daily and a reset at 00:00 UTC.

## Daily Leaderboard Bands

The public table assigns points by final daily leaderboard position, from 10,000 points for first place through smaller awards down to positions 501-1000. The referral-program page repeats the same leaderboard structure for pre-TGE points, while the broader points page says Vibe Points can be earned through trading, referring, and community participation.

Those pages should be read together: the trading-program page gives the rank table; the points page gives the top-level category taxonomy; the referral page shows that similar daily mechanics may also appear in referral contexts.

## Boundary With Dashboard Points

This page should not be used to finalize the onboarding app's internal point rails. The local dashboard distinguishes onboarding, referral, network/trading, and Vibe points, and final TGE settlement language remains owner-controlled.

## Reader Implication

When a trader asks how the trading leaderboard pays points, route here. When a user asks how all points become token claims, route to the TGE settlement and points taxonomy pages until the operator confirms final public wording.
