# Referral Early-Code Scarcity Cohorts

Neelo's referral-economics source explores an early-cohort model where initial referral-code holders receive structurally stronger distribution rights during an access-gated phase.

This is a growth mechanism, not a final public policy by itself. Scarcity can increase launch velocity only if it is paired with transparent issuance and expiry rules.

## What The Model Must Define

The source says any early-code or master-cohort model must define:

- exact cohort size and eligibility path;
- duration of enhanced rights;
- propagation depth limits;
- sunset rules that prevent permanent centralization.

Those details are what keep early access from becoming an unbounded permanent rent. Without them, a scarce-code program can conflict with open participation and referral fairness later.

## Publication Boundary

This page explains the source launch model. It should not publish final cohort size, master-code list, enhanced-right duration, propagation depth, sunset schedule, partner commitments, or retroactive treatment until operator, legal, accounting, and implementation review confirm them.
