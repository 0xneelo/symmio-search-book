# Referral Points As Economic State

The referral-program source makes a strong distinction: points stop being harmless marketing metadata once they can influence tradable outcomes, TGE weighting, transferable claims, or reward-linked assets.

That means points have to be documented as economic state. The source separates fee-linked points from engagement points because the abuse profile is different. Fee-linked points can inherit integrity from real activity if volume, fees, referrals, and LP participation are finalized through verifiable source buckets. Engagement points are easier to farm if they become liquid without controls.

The docs should therefore explain points in three states: provisional off-chain accounting, finalized claimable accounting, and economically active on-chain representation. Each state needs source buckets, eligibility rules, and finalization timing.

## Publication Note

This page does not finalize TGE conversion, transferability, pack eligibility, or reward weighting. It establishes the accounting principle: if points can affect economic outcomes, they need the same seriousness as balances, claims, and attribution.
