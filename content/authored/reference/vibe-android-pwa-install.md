# Installing The Vibe PWA On Android

The official Android path is also a PWA install flow.

The mobile guide says Android users should open Chrome or another browser, visit `app.vibe.trading`, choose Install PWA App, use the browser menu, and add the app to the home screen.

## Browser Install Flow

The public source does not require a native store listing for this path. It describes adding the browser-based PWA to the home screen. That matters for support language: users should be told to install the PWA from the Vibe web app flow, not to search for an unrelated mobile package.

The PWA is a convenience and monitoring surface. The same account, funding, margin, order, and liquidation rules still apply after installation.
