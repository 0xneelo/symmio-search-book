# Vibe TradingView Layouts And Watchlists

Vibe's TradingView controls are meant to make repeated market review less fragile.

The official charting page says users can add indicators, customize chart appearance, save multiple layouts, switch tickers while preserving drawn lines and indicators, and create custom watchlists.

## Why Layout Persistence Matters

The product point is continuity. A trader can study several markets without reconstructing the same chart setup each time. Saved layouts and ticker switching with preserved drawings help users compare markets with less context loss.

The public source does not list every supported indicator, interval, data source, or watchlist limit. Do not turn this page into a full TradingView feature matrix unless a current official source provides that detail.
