# Referral Public And Private Economics Boundary

The rakeback source treats private partner terms as a practical requirement in some growth phases. The control is not to forbid private economics; it is to keep them as explicit overlays rather than hidden side effects inside the public policy.

That distinction matters for trust. A user reading the public table should know what default rules apply. A partner with negotiated terms should know that those terms are separate from the default policy. Operators should be able to reconcile public and private accounting without rewriting the baseline for everyone else.

## Documentation Rule

Public docs should name the default policy and state whether private overlays may exist. They should not disclose confidential partner terms unless approved, and they should not imply that private terms are universal.

The page should also tell readers where to look for policy version, effective date, scope, caps, and stacking behavior once those values are public.
