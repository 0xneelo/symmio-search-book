# Vibe Referral Code Flow

The referral-code page is an onboarding flow, not just a rewards table. When a user creates a Vibe account, the public guide says the user is prompted to enter a friend's referral code, receives a unique referral code that matches the account name, and has an account NFT minted at the same time.

## Entering And Sharing Codes

The current public flow supports entering a friend's code during account creation. It also describes link, QR, and post-account entry paths, with some of those paths marked as coming soon. Referral codes can be shared manually, from My Account as an image to X, and through PnL share cards that include the referral code and QR code.

That mix matters for docs: current manual entry and visible sharing should be documented as product flow; coming-soon link or post-account paths should stay future-facing until the product source says they are live.

## Benefits

The official page separates referrer and referee benefits. Referrers earn Vibe Points for each active referral before TGE and `$VIBE` for each active referral after TGE. Referees can receive up to 50% trading-fee rakeback, with the rakeback page holding the tier table.

The local dashboard's separate referral-depth and historical-accounting question is resolved for v1 public copy: public referral depth is 15 levels, and historical backfill is additive so it never lowers an existing balance. The referral-code page still should not be treated as the full accounting policy for every dashboard metric.

## Reader Implication

When a new user asks how referral codes work, answer with the account-creation flow first: enter a friend's code, receive your own code, and understand where codes can be shared. When the question becomes "how much do I earn?", route to referral commission, rakeback, points, and the 15-level dashboard accounting boundary.

## Publication Boundary

Do not present coming-soon referral-link or post-account entry paths as live until the product source says they are live. Do not collapse referral-code entry, referee rakeback, referrer commission, Vibe Points, dashboard network accounting, and TGE settlement into one blended rewards answer.
